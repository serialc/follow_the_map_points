Real-time replay of GeoJSON point time-series

This application allows you to 'follow' in real-time (or other speed) the path of a provided GeoJSON lat-long point coordinate time-series. This is designed to replay GPS logs, or other movement data, and see it in real-time. All coordinates must be using the WGS 84 geographic coordinate system (lat/long)in GeoJSON formatted point data.


Paste in other GeoJSON features you would like shown. This can be done for multiple GeoJSON feature sets.
