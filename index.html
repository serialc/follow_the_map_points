Real-time replay of GeoJSON points

This application allows you to 'follow' in real-time (or other speed) the path of a provided GeoJSON lat-long point coordinate sequence. This is designed to replay GPS logs and see movement in real-time. All coordinates must be using the WGS 84 geographic coordinate system.
